# 八字命盤分析系統 (BaZi Analyzer)

一個以 Python 打造的八字（四柱）命理排盤與分析 Web 應用。使用者輸入生辰，系統自動排出四柱、判斷日主強弱、推算喜用神與忌神、排列大運與流年，並可一鍵匯出排版精美的 PDF 報告。

## 功能特色

- **自動排四柱**：以農曆干支演算法（sxtwl）將國曆生辰換算為年、月、日、時四柱
- **日主強弱判斷**：依五行力量、月令得氣與否，自動判定身強／身弱
- **喜用神推算**：根據日主強弱，自動推導喜用神與忌神
- **十神標註**：計算各柱天干對日主的十神關係
- **大運排列**：依陰陽順逆排出大運
- **流年逐年分析**：自動偵測流年地支對本命四柱的沖、合關係並分級標示
- **PDF 匯出**：以 reportlab + 繁體中文字型產出可下載的分析報告
- **響應式網頁介面**：桌機與手機皆可使用

## 技術棧

| 層 | 技術 |
|---|---|
| 後端 | Python 3、Flask |
| 排盤演算 | sxtwl（農曆／干支換算）、自製五行/十神/沖合邏輯 |
| PDF 產生 | reportlab（Noto Sans CJK TC 字型） |
| 前端 | 原生 HTML/CSS/JavaScript（fetch API、無框架） |

## 專案結構

## 安裝與執行

```bash
pip install flask sxtwl reportlab
python app.py
# 瀏覽器開啟 http://localhost:5000
```

## 核心模組也可獨立使用

```python
from bazi import BaziChart

chart = BaziChart(1991, 10, 20, 12)   # 年, 月, 日, 時(0-23)
result = chart.full_analysis(2025, 2035)
print(result["pillars"])       # 四柱
print(result["yong_ji"])       # 喜用神/忌神
print(result["liu_nian"])      # 流年逐年
```

## 設計說明與限制

- 起運歲數採簡化值（固定約 8 歲起），實務精算需依出生節氣，屬已知簡化。
- 日主強弱採「幫身力量佔比 + 月令加權」的簡化演算法，與資深命理師的細部判斷可能有出入。
- 大運順逆預設以「男命」邏輯推算。
- 本系統為技術與興趣專案，命理為傳統文化框架，僅供參考。

## v2 新增功能

- **針對性主題分析**：使用者可選擇想問的面向（母親、父親/財、事業、感情、財運、才華、健康、學業），系統自動找出對應的「十神」在命盤中的位置，判斷喜忌，並掃描流年中對該星的沖／合，列出需注意的年份。這是命理「看什麼問題、找什麼星」邏輯的程式化。
- **術語即時解釋**：網頁上專有名詞（喜用神、身弱、十神、沖、合等，共 25 項）滑鼠移過即顯示白話說明。
- **白話解讀區**：自動用大白話解釋整張命盤，網頁與 PDF 皆有。
